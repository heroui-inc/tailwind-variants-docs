## How do I enable autocompletion in my IDE?

To enable autocompletion inside `tv` in your favourite IDE, use the same method as for `cva` which is explained [here](https://github.com/joe-bell/cva/blob/f45fdafc2031e82e10ef4aba3bd8e64351bc7b2e/docs/pages/docs/installation.mdx#intellisense). This means adding `classRegex` to your Tailwind CSS settings. In WebStorm, you can find this under the Tailwind CSS page in the Settings tab. In Visual Studio Code, add it to your [`settings.json`](https://code.visualstudio.com/docs/getstarted/settings) file.

### Visual Studio Code
```json
{
  "tailwindCSS.experimental.classRegex": [
    ["tv\\(([^)]*)\\)", "[\"'`]([^\"'`]*).*?[\"'`]"]
  ]
}
```

### WebStorm
```json
{
  "experimental": {
    "classRegex": [["tv\\(([^)]*)\\)", "[\"'`]([^\"'`]*).*?[\"'`]"]]
  }
}
```
