# Upgrade guide

Upgrade `tailwind-variants` to `v1.x`.

### Tailwindcss v4 Support

`tailwind-variants` `v1.x` now supports `Tailwindcss v4`. You can use all the new features and improvements from `Tailwindcss v4` in your projects.

### Run the upgrade script

To upgrade your project to `v1.x`, you can run the upgrade script.

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tab>

    ```bash copy
    npm install tailwind-variants@latest
    ```

  </Tab>
  <Tab>

    ```bash copy
    yarn upgrade tailwind-variants@latest
    ```

  </Tab>
  <Tab>

    ```bash copy
    pnpm upgrade tailwind-variants@latest
    ```

  </Tab>
</Tabs>
